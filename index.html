OSI-MODEL

Please do not throw sausage pizza away
Physical layer: the trafficknis in bits -devices include data cable, hubs and repeaters.
Data link layer; the traffic is in frames - devices include switches
Network layer; data trafffic is in packets - devices include routers and firewalls
Transport layer ; traffic is in segments - TCP/UDP
Session layer - management of sessions
Presentation layer - jpeg
Application layer - HTTP, SMTP

Each of the above layer is responsible for managing communication between specific componentsin the network. Each layer is said to be independent such that it's actions are not affected by the actions perfomed at the other layers.
